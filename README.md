# Task Manager App

A simple full-stack task manager using React, Node.js, Express.

## Features

- View and add tasks
- Simple REST API
- React frontend with basic UI

## Tech Stack

- Frontend: React
- Backend: Node.js, Express

## How to Run

### Server
1. Navigate to `server/`
2. Run `npm install`
3. Run `node index.js`

### Client
1. Navigate to `client/`
2. Run `npm install`
3. Run `npm start`
